> Manage your Notion&#39;s tasks inside GitHub

## Usage

Install:

```sh
yarn add danger-plugin-notion-tasks --dev
```

At a glance:

```js
// dangerfile.js
import notionTasks from 'danger-plugin-notion-tasks'

notionTasks()
```
